k8s yaml defination

k8s yaml defination

YAML fileds

一个常见的pod定义如下：

apiVersion: v1
kind: Pod
metadata:
  name: mywebserver
spec:
  containers:
    - name: mywebserver
      image: nginx

k8s里的任何对象，都包括以下四个字段：

Key Description
apiVersion Version of API
kind Kind of object you want to scale
Metadata name Name of object that uniquely identifies it.
spec Desired state of the object.
  • apiVersion表示api版本，不同的对象版本可能不同。例如v1, v1beta等
  • kind表示对象类型，例如Deployment, Pod, Service, Secret....

  • Metadata里一般声明对象的名称、标签等。

  • spec区域是Desired state of the object. 是对象的核心描述。
